ImpactJS Box2D 2.2.1 Plugin

----------------

This port of Box2D was compiled from C++ to JavaScript using Emscripten.

I've added several improvements to Impact entities such as [.touches](http://impactjs.com/documentation/class-reference/entity#touches) support.
